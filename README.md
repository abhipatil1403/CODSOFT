## Project Highlights

### 1. KAAR Studio — Full-Stack E-Commerce App
A curated, premium e-commerce platform specializing in luxury Indian handicrafts and textiles, built with a minimalist, high-end aesthetic.
* **Tech Stack:** React (Vite), Node.js, Express.js, MongoDB Atlas (Cloud Database).
* **Key Features:**
  * User Authentication (Registration & Login).
  * Cloud-based product catalog database.
  * Interactive Cart with custom quantity steppers.
  * Dedicated checkout flow tailored for the Indian context (RuPay card visualizer and simulated BHIM UPI QR scanner).
  * Fully responsive mobile layouts and custom glassmorphism design.
* **Live Deployment:**
  * **Frontend:** Hosted on Netlify at [https://kaar-ecommerce.netlify.app/](https://kaar-ecommerce.netlify.app/)
  * **Backend:** Hosted on Render at [https://kaar-backend-b3ao.onrender.com/](https://kaar-backend-b3ao.onrender.com/)

### 2. TaskFlow — Project Management Tool
An interactive task management dashboard featuring a dynamic Kanban Board for organizing projects.
* **Tech Stack:** React (Vite), Supabase (PostgreSQL Database & Realtime Auth).
* **Key Features:**
  * User Sign Up, Login, and Session management.
  * Interactive Kanban Board with column-switching, sorting, and task drag-and-drop workflows.
  * Task creation, updates, completion states, and prioritization labels.
  * Real-time persistence with Supabase PostgreSQL backend database.
